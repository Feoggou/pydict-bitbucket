rerun

Definitions
verb (riːˈrʌn ) -runs, -running, -ran or run (transitive)1. to broadcast or put on (a film, play, series, etc) again2. to run (a race, etc) againnoun (ˈriːˌrʌn )3. a film, play, series, etc, that is broadcast or put on again; repeat4. a race that is run again5. computing the repeat of a part of a computer program
