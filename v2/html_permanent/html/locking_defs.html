locking

(ˈlɒkɪŋ)
Definitions
adjective1. fastening with a lock; able to be locked ⇒ a locking petrol capnoun2. computing the immobilizing of a screen, for security 3. the system for locking the doors of a vehicle ⇒ features such as electric windows, sun-roofs, metallic paint and remote locking. See also central locking
