laudanum

(ˈlɔːdənəm )
Definitions
noun1. a tincture of opium2. (formerly) any medicine of which opium was the main ingredient
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C16: New Latin, name chosen by Paracelsus for a preparation probably containing opium, perhaps based on labdanum
