capo1

(ˈkeɪpəʊ , ˈkæpəʊ )
Definitions
Collins English Dictionary
noun plural -posa device fitted across all the strings of a guitar, banjo, etc, so as to raise the pitch of each string simultaneously. Also called: capo tasto (ˈkæpəʊ ˈtæstəʊ ) Compare barré
Word origin
from Italian capo tasto head stop
