conjuration

(ˌkɒndʒʊˈreɪʃən )
Definitions
Collins English Dictionary
noun1. a magic spell; incantation2. a less common word for conjuring3. archaic supplication; entreaty
Example sentences containing 'conjuration'
Had they discovered some conjuration previously unknown to them, which opened doors where there had only been solid walls? Clive Barker EVERVILLEThe abracadabra of the mages operating on their minds, some devilish conjuration. Robert Silverberg LORD PRESTIMION
