postbox

(ˈpəʊstˌbɒks )
Definitions
Collins English Dictionary
nounmainly British a box into which mail is put for collection by the postal service. Also called: letter box

Definitions
COBUILD Advanced British English Dictionary
countable nounA postbox is a metal box in a public place, where you put letters and packets to be collected. They are then sorted and delivered. Compare letterbox. [British]regional note:   in AM, use mailbox
