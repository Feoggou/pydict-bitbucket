intercourse

(ˈɪntəˌkɔːs )
Definitions
noun1. communication or exchange between individuals; mutual dealings2. See sexual intercourse
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C15: from Medieval Latin intercursus business, from Latin intercurrere to run between, from currere to run
