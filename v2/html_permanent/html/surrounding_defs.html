surrounding

(səˈraʊndɪŋ ; sərˈraʊndɪŋ )
Definitions
Webster’s New World College Dictionary
noun1.  that which surrounds2.  [pl.] the things, conditions, influences, etc. that surround a given place or person; environment adjective3.  that surrounds
