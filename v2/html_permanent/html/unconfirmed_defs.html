(ʌnkənfɜːʳmd )
Definitions
adjectiveIf a report or a rumour is unconfirmed, there is no definite proof as to whether it is true or not. OPPOSITE: confirmed ⇒ There are unconfirmed reports of several small villages buried by mudslides.
