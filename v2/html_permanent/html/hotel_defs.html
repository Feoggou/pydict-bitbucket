hotel

(həʊˈtɛl )
Definitions
nouna commercially run establishment providing lodging and usually meals for guests, and often containing a public bar
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C17: from French hôtel, from Old French hostel; see hostel
