Definitions
Collins English Dictionary
nouna number or quantity that when multiplied by itself gives a given number or quantity ⇒ 2 is a square root of 4, usually written √4 or 41/2

Word forms: square roots
Definitions
COBUILD Advanced British English Dictionary
countable nounThe square root of a number is another number which produces the first number when it is multiplied by itself. For example, the square root of 16 is 4.
