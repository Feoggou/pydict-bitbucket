legible

(ˈlɛdʒəbəl )
Definitions
adjective1. (of handwriting, print, etc) able to be read or deciphered2. able to be discovered; discernible
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

legibility (ˌlegiˈbility) or legibleness (ˈlegibleness) nounlegibly (ˈlegibly) adverb

Word Origin

C14: from Late Latin legibilis, from Latin legere to read
