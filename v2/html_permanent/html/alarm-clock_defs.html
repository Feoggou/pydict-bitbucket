Definitions
COBUILD Key Words for Hospitality
noun(Hospitality (hotel): Hotel room, bedroom) An alarm clock is a clock that you can set to make a noise so that it wakes you up at a particular time. ⇒ Is there an alarm clock in the room? ⇒ The alarm clock doesn't work and I overslept. ⇒ Many hotels offer alarm clocks and wake-up calls to guests who need to rise early.Words used with 'alarm clock'If you set an alarm clock, you press buttons on it so that it will make a noise at a particular time, and if an alarm clock goes off, it starts to make a noise.
