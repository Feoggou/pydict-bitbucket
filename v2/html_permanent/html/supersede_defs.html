supersede

(ˌsuːpəˈsiːd )

Definitions

Collins English Dictionary
verb (transitive)1. to take the place of (something old-fashioned or less appropriate); supplant2. to replace in function, office, etc; succeed3. to discard or set aside or cause to be set aside as obsolete or inferior

Derived Forms

ˌsupersedable (ˌsuperˈsedable) adjectiveˌsupersedence (ˌsuperˈsedence) nounˌsuperseder (ˌsuperˈseder) nounsupersedure (ˌsuːpəˈsiːdʒə ) nounsupersession (ˌsuːpəˈsɛʃən ) noun

Word Origin

C15: via Old French from Latin supersedēre to sit above, from super- + sedēre to sit
