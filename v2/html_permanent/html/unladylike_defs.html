unladylike

(ʌnˈleɪdɪlaɪk)
Definitions
adjective1. that does not behave in a way considered proper for a lady2. that is not considered proper for a lady ⇒ It's unladylike to yawn.
Collins English Dictionary. Copyright © Harper Collins Publishers
Example sentences containing 'unladylike'
`Surely, Susan, you're not going to do anything so - so unladylike as to throw that at me?
