flue1

(fluː )
Definitions
noun1. a shaft, tube, or pipe, esp as used in a chimney, to carry off smoke, gas, etc2. music the passage in an organ pipe or flute within which a vibrating air column is set up. See also flue pipe
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin
