immemorial

(ˌɪmɪˈmɔːrɪəl )
Definitions
adjectiveoriginating in the distant past; ancient (postpositive in the phrase time immemorial)
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

immemorially (ˌimmeˈmorially) adverb

Word Origin

C17: from Medieval Latin immemoriālis, from Latin im- (not) + memoria memory

Definitions
1. →  since time immemorial/from time immemorial2. adjective [usu ADJ n]You use immemorial to indicate that something has existed for many centuries. [literary] ancient ⇒ ...a modern version of an immemorial myth.
