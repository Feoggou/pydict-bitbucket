directory

(dɪˈrɛktərɪ , -trɪ, daɪ-)
Definitions
Collins English Dictionary
noun plural -ries1. a book, arranged alphabetically or classified by trade listing names, addresses, telephone numbers, etc, of individuals or firms2. a book or manual giving directions3. a book containing the rules to be observed in the forms of worship used in churches4. a less common word for directorate (sense 2)5. computing an area of a disk, Winchester disk, or floppy disk that contains the names and locations of files currently held on that diskadjective6. directing

(daɪrektəri , dɪr- )
Word forms: directories
Definitions
COBUILD Advanced British English Dictionary
1. countable noun [oft n N]A directory is a book which gives lists of facts, for example people's names, addresses, and telephone numbers, or the names and addresses of business companies, usually arranged in alphabetical order. ⇒ ...a telephone directory.2. countable nounA directory is an area of a computer disk which contains one or more files or other directories. [computing] ⇒ This option lets you create new files or directories.3. countable nounOn the World Wide Web, a directory is a list of the subjects that you can find information on. [computing] ⇒ Yahoo is the oldest and best-known Web directory service.
