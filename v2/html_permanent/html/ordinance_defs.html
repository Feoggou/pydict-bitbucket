ordinance

(ˈɔːdɪnəns )
Definitions
nounan authoritative regulation, decree, law, or practice
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C14: from Old French ordenance, from Latin ordināre to set in order
