password

(ˈpɑːsˌwɜːd )
Definitions
Collins English Dictionary
noun1. a secret word, phrase, etc, that ensures admission or acceptance by proving identity, membership, etc2. an action, quality, etc, that gains admission or acceptance3. a sequence of characters used to gain access to a computer system

Definitions
COBUILD Advanced British English Dictionary
countable nounA password is a secret word or phrase that you must know in order to be allowed to enter a place such as a military base, or to be allowed to use a computer system. ⇒ Advance and give the password. ⇒ No-one could use the computer unless they had a password.
Example sentences containing 'password'
Hannah's simple password "Today" had opened up everything on the hard drive. Elizabeth Lowell PEARL COVE
