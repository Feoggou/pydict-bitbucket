barré

(ˈbæreɪ )
Definitions
Collins English Dictionary
noun1. the act of laying the index finger over some or all of the strings of a guitar, lute, or similar instrument, so that the pitch of each stopped string is simultaneously raised. Compare capo12. the playing of chords in this mannerverb3. to execute (chords) in this manneradverb4. by using the barré
Word origin
C19: from French, from barrer bar1
