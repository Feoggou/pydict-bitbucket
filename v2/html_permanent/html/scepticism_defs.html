(skeptɪsɪzəm )
Definitions
COBUILD Advanced British English Dictionary
regional note:   in AM, use skepticismuncount nounScepticism is great doubt about whether something is true or useful. ⇒ There was considerable scepticism about the Chancellor's forecast of a booming economy. ⇒ The report has inevitably been greeted with scepticism.
