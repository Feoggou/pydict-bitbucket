Definitions
COBUILD Advanced British English Dictionary
1. adjectiveMono is used to describe a system of playing music in which all the sound is directed through one speaker only. Compare stereo. ⇒ This model has a mono soundtrack.2. uncount nounMono is the same as mononucleosis. [US, informal]
