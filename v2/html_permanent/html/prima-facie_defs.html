prima facie

(ˈpraɪmə ˈfeɪʃɪ )
Definitions
Collins English Dictionary
unknownat first sight; as it seems at first

Word Origin

C15: from Latin, from prīmus first + faciēs face

prima facie

Powered by COBUILD
(praɪmə feɪʃi )
Definitions
COBUILD Advanced British English Dictionary
adjective [usu ADJ n]Prima facie is used to describe something which appears to be true when you first consider it. [formal] ⇒ There was a prima facie case that a contempt of court had been committed.
Example sentences containing 'prima facie'
If we discover enough proof to make a prima facie case of sabotage against persons unknown, then the RCMP will have to take notice.
