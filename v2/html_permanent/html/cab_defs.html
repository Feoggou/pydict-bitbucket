cab1

(kæb )
Definitions
Collins English Dictionary
noun1. a. a taxib. (as modifier) ⇒ a cab rank2. the enclosed compartment of a lorry, locomotive, crane, etc, from which it is driven or operated3. (formerly) a light horse-drawn vehicle used for public hire4. See first cab off the rank
Word origin
C19: shortened from cabriolet
