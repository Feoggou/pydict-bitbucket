compile

(kəmˈpaɪl )
Definitions
Collins English Dictionary
verb (transitive)1. to make or compose from other materials or sources ⇒ to compile a list of names2. to collect or gather for a book, hobby, etc3. computing to create (a set of machine instructions) from a high-level programming language, using a compiler
Word origin
C14: from Latin compīlāre to pile together, plunder, from com- together + pīlāre to thrust down, pack
