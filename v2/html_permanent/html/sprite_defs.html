sprite

(spraɪt )
Definitions
Collins English Dictionary
noun1. (in folklore) a nimble elflike creature, esp one associated with water2. a small dainty person3. an icon in a computer game which can be manoeuvred around the screen by means of a joystick, etc

Word Origin

C13: from Old French esprit, from Latin spīritus spirit1
