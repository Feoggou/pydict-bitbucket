colonnade

(ˌkɒləˈneɪd )
Definitions
Collins English Dictionary
noun1. a set of evenly-spaced columns2. a row of regularly spaced trees
Derived forms
ˌcolonnaded (ˌcolonˈnaded) adjective
Word origin
C18: from French, from colonne column; on the model of Italian colonnato, from colonna column
