all along

Powered by COBUILD
Definitions
COBUILD Advanced British English Dictionary
phraseIf something has been true or been present all along, it has been true or been present throughout a period of time. ⇒ I've been fooling myself all along. ⇒ I think she had been planning all along to leave Hungary. →  along
