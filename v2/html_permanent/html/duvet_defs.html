duvet

(ˈduːveɪ )
Definitions
Collins English Dictionary
noun1. another name for continental quilt2. Also called: duvet jacket a down-filled jacket used esp by mountaineers
Word origin
C18: from French, from earlier dumet, from Old French dum down2
