forgetful

(fəˈɡɛtfʊl )
Definitions
adjective1. tending to forget2. (often postpositive; foll by of) inattentive (to) or neglectful (of)3. poetic causing loss of memory
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

forgetfully (forˈgetfully) adverbforgetfulness (forˈgetfulness) noun
