logarithm

(ˈlɒɡəˌrɪðəm )
Definitions
nounthe exponent indicating the power to which a fixed number, the base, must be raised to obtain a given number or variable. It is used esp to simplify multiplication and division: if ax = M, then the logarithm of M to the base a (written logaM) is x. Often shortened to: log See also common logarithm, natural logarithm
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C17: from New Latin logarithmus, coined 1614 by John Napier, from Greek logos ratio, reckoning + arithmos number
