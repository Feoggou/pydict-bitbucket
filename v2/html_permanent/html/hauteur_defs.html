hauteur

(əʊˈtɜː )
Definitions
nounpride; haughtiness
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C17: from French, from haut high; see haughty

Definitions
uncount nounYou can use hauteur to describe behaviour which you think is proud and arrogant. [formal, disapproval] ⇒ Once, she had been put off by his hauteur.
