decimal system

Definitions
Collins English Dictionary
noun1. the number system in general use, having a base of ten, in which numbers are expressed by combinations of the ten digits 0 to 92. a system of measurement, such as the metric system, in which the multiple and submultiple units are related to a basic unit by powers of ten
