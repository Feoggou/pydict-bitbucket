translation

(trænsˈleɪʃən , trænz-)
Definitions
Collins English Dictionary
noun1. something that is or has been translated, esp a written text2. the act of translating or the state of being translated3. mathematics a transformation in which the origin of a coordinate system is moved to another position so that each axis retains the same direction or, equivalently, a figure or curve is moved so that it retains the same orientation to the axes

Derived Forms

translational (transˈlational) adjective
