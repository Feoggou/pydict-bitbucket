aspirated

(ˈæspɪˌreɪtɪd)
Definitions
Collins English Dictionary
adjective1. phonetics (of a stop) articulated with some force, so that breath escapes with audible friction as the stop is released ⇒ an aspirated bilabial unvoiced stop, ⇒ aspirated consonants2. mechanical engineering (of an internal-combustion engine) supplied with air in the stated way ⇒ a naturally aspirated engine, ⇒ features that set the car apart from the normally aspirated model
Example sentences containing 'aspirated'
