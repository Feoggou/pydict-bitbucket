quittance

(ˈkwɪtəns )
Definitions
noun1. release from debt or other obligation2. a receipt or other document certifying this
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C13: from Old French, from quitter to release from obligation; see quit
