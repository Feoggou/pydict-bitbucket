run out

Definitions
verb (adverb)1. (intransitive; often foll by of) to exhaust (a supply of something) or (of a supply) to become exhausted2. (intransitive) to expire; become no longer valid ⇒ my passport has run out3. See run out on4. (transitive) cricket to dismiss (a running batsman) by breaking the wicket with the ball, or with the ball in the hand, while he is out of his groundnoun runout5. cricket dismissal of a batsman by running him out6. mechanical engineering an imperfection of a rotating component so that not all parts revolve about their intended axes relative to each other

run out

Powered by COBUILD
for learners of English
Definitions
1. phrasal verbIf you run out of something, you have no more of it left. [V P + of] ⇒ They have run out of ideas. [V P of n] ⇒ We're running out of time. [V P of n] ⇒ By now the plane was running out of fuel. [V P] ⇒ We had lots before but now we've run out.2. → to run out of steam3. phrasal verbIf something runs out, it becomes used up so that there is no more left. [V P] ⇒ Conditions are getting worse and supplies are running out. [V P] ⇒ Time is running out.4. phrasal verbWhen a legal document runs out, it stops being valid. [V P] ⇒ When the lease ran out the family moved to Campigny. [V P] ⇒ ...the day my visa ran out. →  run
