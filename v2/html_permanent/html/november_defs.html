November

(nəʊˈvɛmbə )
Definitions
noun1. the eleventh month of the year, consisting of 30 days2. communications a code word for the letter n
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C13: via Old French from Latin: ninth month, from novem nine
