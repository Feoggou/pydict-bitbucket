febrifuge

(ˈfɛbrɪˌfjuːdʒ )
Definitions
noun1. any drug or agent for reducing feveradjective2. serving to reduce fever
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

febrifugal (fɪˈbrɪfjʊɡəl , ˌfɛbrɪˈfjuːɡəl ) adjective

Word Origin

C17: from Medieval Latin febrifugia feverfew; see febri-, -fuge
