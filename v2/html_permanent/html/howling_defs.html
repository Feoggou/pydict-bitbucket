howling

(ˈhaʊlɪŋ )
Definitions
adjective(prenominal) informal (intensifier) ⇒ a howling success, ⇒ a howling error
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

howlingly (ˈhowlingly) adverb
