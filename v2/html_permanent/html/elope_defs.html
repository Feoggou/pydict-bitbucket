elope

(ɪˈləʊp )
Definitions
verb(intransitive) to run away secretly with a lover, esp in order to marry
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

elopement (eˈlopement) nouneloper (eˈloper) noun

Word Origin

C16: from Anglo-French aloper, perhaps from Middle Dutch lōpen to run; see lope
