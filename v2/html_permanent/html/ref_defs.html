Definitions
1. Ref. is an abbreviation for reference. It is written in front of a code at the top of business letters and documents. The code refers to a file where all the letters and documents about the same matter are kept. [business] ⇒ Our Ref: JAH/JW.2. countable nounThe ref in a sports game, such as football or boxing, is the same as the referee. [informal] ⇒ The ref gave a penalty and Zidane scored.
