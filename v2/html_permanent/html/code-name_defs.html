code name

or codename (ˈkəʊdˌneɪm)
Definitions
Collins English Dictionary
noun1. (esp in military and police use) a word used to identify a classified plan, operation, etc ⇒ One of their informers was working under the code name Czerny., ⇒ The operation was given the code name Dynamo.verb code-name (transitive)2. to assign a code name to (a person, group, or operation) ⇒ The operation was code-named Moonlight Sonata.

Word forms: code names , code naming , code named
Definitions
COBUILD Advanced British English Dictionary
1. countable noun [usu N n]A code name is a name used for someone or something in order to keep their identity secret. ⇒ One of their informers was working under the code name Czerny. ⇒ The operation was given the code name Dynamo.2. verb [usu passive]If a military or police operation is code-named something, it is given a name which only the people involved in it know. [be V-ed n] ⇒ The operation was code-named Moonlight Sonata. [V-ed] ⇒ ...a military contingent, code-named Sparrowhawk.
