consanguineous

(ˌkɑnsæŋˈgwɪniəs ; ˌkɑnsænˈgwɪniəs )
Definitions
Webster’s New World College Dictionary
adjective having the same ancestor; closely related: also conˈsanguine (ˌkɑnˈsæŋgwɪn )
