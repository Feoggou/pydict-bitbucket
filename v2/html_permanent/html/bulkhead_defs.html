bulkhead

(ˈbʌlkˌhɛd )
Definitions
Collins English Dictionary
noun1. any upright wall-like partition in a ship, aircraft, vehicle, etc2. a wall or partition built to hold back earth, fire, water, etc
Word origin
C15: probably from bulk projecting framework, from Old Norse bálkr partition + head
