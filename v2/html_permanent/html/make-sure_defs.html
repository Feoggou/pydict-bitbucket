make sure

Definitions
Collins English Dictionary
a. (takes a clause as object) to make certain; ensureb. (foll by of) to establish or confirm power or possession (over)▷ See sure

Definitions
COBUILD Advanced British English Dictionary
phraseIf you make sure that something is done, you take action so that it is done. ⇒ Make sure that you follow the instructions carefully. ⇒ He wants to make sure that schools are committed to providing alternative education. →  sure

make sure

Powered by COBUILD
Definitions
COBUILD Advanced British English Dictionary
phraseIf you make sure that something is the way that you want or expect it to be, you check that it is that way. make certain, check ⇒ He looked in the bathroom to make sure that he was alone. ⇒ Before you cut the cloth, make sure that the pattern matches up on both edges. →  sure
