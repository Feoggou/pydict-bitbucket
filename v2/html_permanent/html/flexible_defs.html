flexible

(ˈflɛksɪbəl )
Definitions
adjective1. Also: flexile (ˈflɛksaɪl ) able to be bent easily without breaking; pliable2. adaptable or variable ⇒ flexible working hours3. able to be persuaded easily; tractable
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

flexibility (ˌflexiˈbility) or flexibleness (ˈflexibleness) nounflexibly (ˈflexibly) adverb

Definitions

[approval] ⇒ Look for software that's flexible enough for a range of abilities. ⇒ ...flexible working hours.flexibly (fleksɪbli ) graded adverb [ADV with v] ⇒ It would seem more sensible to apply standards flexibly rather than rigidly.flexibility uncount noun adaptability ⇒ The flexibility of distance learning would be particularly suited to busy managers.
