Definitions
Collins English Dictionary
nounSee Mary Magdalene

Magdalene - also, for 2,

(ˈmægdələn ; ˈmægdəlɪn ; ˈmægdəˌlin ; ˌ mæg dəˈlinə )
Definitions
Webster’s New World College Dictionary
noun Obsolete1.  a feminine name: dim. Lena; var. Madeline, Madelyn2.  Bible Mary Magdalene: Luke 8:2 (identified with the repentant woman in Luke 7:37)3.  [m-]; Archaic a reformed and repentant prostitute4.  [m-]; British, Archaic a reformatory for prostitutes

Word Origin
