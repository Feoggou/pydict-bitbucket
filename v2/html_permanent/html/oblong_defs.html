oblong

(ˈɒbˌlɒŋ )
Definitions
adjective1. having an elongated, esp rectangular, shapenoun2. a figure or object having this shape
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C15: from Latin oblongus, from ob- (intensive) + longus long1
