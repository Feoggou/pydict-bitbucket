pervious

(ˈpɜːvɪəs )
Definitions
Collins English Dictionary
adjective1. able to be penetrated; permeable2. receptive to new ideas; open-minded

Derived Forms

perviously (ˈperviously) adverbperviousness (ˈperviousness) noun

Word Origin

C17: from Latin pervius, from per- (through) + via a way
