Ecclesiastes

(ɪˌkliːzɪˈæstiːz )
Definitions
noun(functioning as singular) a book of the Old Testament, probably written about 250 bc
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

via Late Latin, from Greek ekklēsiastēs member of the assembly; see ecclesia
Example sentences containing 'Ecclesiastes'
'Whatsoever thy die findeth for you to do, do it with all thy might' - Ecclesiastes (Revised Version). Luke Rhinehart THE BOOK OF THE DIE
