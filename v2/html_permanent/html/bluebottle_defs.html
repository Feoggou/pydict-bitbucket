bluebottle

(ˈbluːˌbɒtəl )
Definitions
Collins English Dictionary
noun1. another name for the blowfly2. any of various blue-flowered plants, esp the cornflower3. British an informal word for a policeman4. Australian and New Zealand an informal name for Portuguese man-of-war
