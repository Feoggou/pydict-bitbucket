airship

(ˈɛəˌʃɪp )
Definitions
Collins English Dictionary
nouna lighter-than-air self-propelled craft. Also called: dirigible, zeppelin

(eəʳʃɪp )

Definitions
COBUILD Advanced British English Dictionary
countable nounAn airship is an aircraft that consists of a large balloon which is filled with gas and is powered by an engine. It has a section underneath for passengers.
