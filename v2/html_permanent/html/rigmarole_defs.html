rigmarole

(ˈrɪɡməˌrəʊl ) or rigamarole
Definitions
noun1. any long complicated procedure2. a set of incoherent or pointless statements; garbled nonsense
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C18: from earlier ragman roll a list, probably a roll used in a medieval game, wherein various characters were described in verse, beginning with Ragemon le bon Ragman the good
