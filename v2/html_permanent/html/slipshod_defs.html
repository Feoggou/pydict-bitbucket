slipshod

(ˈslɪpˌʃɒd )

Definitions

Collins English Dictionary
adjective1. (of an action) negligent; careless2. (of a person's appearance) slovenly; down-at-heel

Derived Forms

slipˌshoddiness (ˈslipˌshoddiness) or slipˌshodness (ˈslipˌshodness) noun

Word Origin

C16: from slip1 + shod

Definitions
COBUILD Advanced British English Dictionary
graded adjective & adjective [usu ADJ n]If something is slipshod, it has been done in a careless way. ⇒ The hotel had always been run in a slipshod way.
