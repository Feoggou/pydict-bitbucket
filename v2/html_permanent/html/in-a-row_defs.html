Definitions
phraseIf something happens several times in a row, it happens that number of times without a break. If something happens several days in a row, it happens on each of those days. ⇒ They have won five championships in a row. ⇒ If I'm inside for three days in a row, I go crazy. ⇒ It is the sixth month in a row in which imports have fallen. →  row
