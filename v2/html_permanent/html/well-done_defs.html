well-done

Definitions
adjective (well done when postpositive)1. (of food, esp meat) cooked thoroughly2. made or accomplished satisfactorily
Collins English Dictionary. Copyright © Harper Collins Publishers

well done in Hospitality

Powered by COBUILD
(wɛl dʌn)
Definitions
adjective(Hospitality (hotel): Food and drink, meat) If something that you have cooked, especially meat, is well done, it has been cooked thoroughly. ⇒ Could I have the steak very well done please? ⇒ Allow the lamb an extra 10-15 minutes to cook if the customer wants it well done. ⇒ Well done steak will be very firm to the touch.
Collins English Dictionary. Copyright © Harper Collins Publishers
