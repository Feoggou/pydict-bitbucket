inextricable

(ˌɪnɛksˈtrɪkəbəl )
Definitions
adjective1. not able to be escaped from ⇒ an inextricable dilemma2. not able to be disentangled, etc ⇒ an inextricable knot3. extremely involved or intricate
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

inextricability (ˌinextricaˈbility) or inextricableness (ˌinexˈtricableness) nouninextricably (ˌinexˈtricably) adverb
