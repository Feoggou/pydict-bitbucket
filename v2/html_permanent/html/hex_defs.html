hex1

(hɛks )
Definitions
nouna. short for hexadecimalb. (as modifier) ⇒ hex code

hex2 - informal

(hɛks )
Definitions
verb1. (transitive) to bewitchnoun2. an evil spell or symbol of bad luck3. a witch

Derived Forms

Word Origin
