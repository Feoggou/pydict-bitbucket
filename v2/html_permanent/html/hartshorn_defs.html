hartshorn

(ˈhɑrtsˌhɔrn ) in American English
Definitions
noun1.  a hart's antler2.  Rare ammonium carbonate, used in smelling salts; sal volatileso called because formerly obtained from deers' antlers

hartshorn

(ˈhɑːtsˌhɔːn )
Definitions
nounan obsolete name for sal volatile (sense 2)
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin
