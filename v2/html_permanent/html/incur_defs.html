incur

(ɪnˈkɜː )
Definitions
verb -curs, -curring or -curred (transitive)1. to make oneself subject to (something undesirable); bring upon oneself2. to run into or encounter
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

incurrable (inˈcurrable) adjective

Word Origin

C16: from Latin incurrere to run into, from currere to run
