ipso facto

Powered by COBUILD
(ɪpsoʊ fæktoʊ ) for learners of English
Definitions
adverbIf something is ipso facto true, it must be true, because of a fact that has been mentioned. ⇒ If a crime occurs then there is, ipso facto, a guilty party.
COBUILD Advanced English Dictionary. Copyright © Harper Collins Publishers
