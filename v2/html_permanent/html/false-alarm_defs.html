false alarm

Definitions
noun1. a needless alarm given in error or with intent to deceive2. an occasion on which danger is perceived but fails to materialize

Word forms: false alarms
Definitions
countable nounWhen you think something dangerous is about to happen, but then discover that you were mistaken, you can say that it was a false alarm. ⇒ ...a bomb threat that turned out to be a false alarm.
