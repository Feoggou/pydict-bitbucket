eyebrow

(ˈaɪˌbraʊ )
Definitions
noun1. the transverse bony ridge over each eye2. the arch of hair that covers this ridge ▶ Related adjective: superciliary3. See raise an eyebrow
Collins English Dictionary. Copyright © Harper Collins Publishers
