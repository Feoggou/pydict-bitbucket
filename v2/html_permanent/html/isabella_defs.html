Isabella1

(ˌɪzəˈbɛlə ) or Isabel (ˈɪzəˌbɛl )
Definitions
nouna. a greyish-yellow colourb. Also: Isabelline (ˌɪzəˈbɛliːn ) (as adjective) ⇒ an Isabella mohair coat
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C17: from the name Isabella; original reference uncertain

Isabella2

(ˌɪzəˈbɛlə )
Definitions
noun original name Elizabeth Farnese. 1692–1766, second wife (1714–46) of Philip V of Spain and mother of Charles III of Spain
Collins English Dictionary. Copyright © Harper Collins Publishers
