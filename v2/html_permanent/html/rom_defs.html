ROM

Definitions
abbreviation(Electrical engineering: Semiconductor and electronic circuitry) ROM is the permanent part of a computer's memory, where stored information can be read but not changed. ⇒ The monitor program of a single-board microcomputer is generally stored in the ROM. ⇒ The processor's ROM is programmed with preset information that is permanently programmed with core functions in order to facilitate processor communication with the data bus. ⇒ ROM is the permanent part of a computer's memory, where stored information can be read but not changed.
