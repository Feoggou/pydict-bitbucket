hands

(hændz )
Definitions
plural noun1. power or keeping ⇒ your welfare is in his hands2. Also called: handling soccer the infringement of touching the ball with any part of the hand or arm3. See change hands4. See clean hands5. See hands down6. See hands off7. See hands up!8. See have one's hands full9. See have one's hands tied10. See in good hands11. See join hands12. See lay hands on13. See off one's hands14. See on one's hands15. See out of one's hands16. See throw up one's hands17. See wash one's hands of
