Romeo

(ˈrəʊmɪəʊ )
Definitions
noun1. plural -os an ardent male lover2. communications a code word for the letter r
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

from the hero of Shakespeare's Romeo and Juliet (1594)
