Asiatic

(ˌeɪʃɪˈætɪk , -zɪ-)
Definitions
Collins English Dictionary
noun, adjectivea word formerly used for Asian▶ USAGE Asiatic is the correct word to use in scientific language when talking about flora, fauna and anthropology, for instance Asiatic lion. Using it as a noun is offensive and old-fashioned
