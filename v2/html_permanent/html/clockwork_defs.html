clockwork

(ˈklɒkˌwɜːk )
Definitions
Collins English Dictionary
noun1. the mechanism of a clock2. any similar mechanism, as in a wind-up toy3. See like clockwork

Definitions
COBUILD Advanced British English Dictionary
1. adjective [ADJ n]A clockwork toy or device has machinery inside it which makes it move or operate when it is wound up with a key. ⇒ ...a clockwork train-set.2. →  like clockwork
