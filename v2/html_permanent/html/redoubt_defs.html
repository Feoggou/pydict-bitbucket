redoubt

(rɪˈdaʊt )
Definitions
noun1. an outwork or detached fieldwork defending a pass, hilltop, etc2. a temporary defence work built inside a fortification as a last defensive position
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C17: via French from obsolete Italian ridotta, from Medieval Latin reductus shelter, from Latin redūcere to withdraw, from re- + dūcere to lead
