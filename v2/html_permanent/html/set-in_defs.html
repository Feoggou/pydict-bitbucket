set in

Definitions

Collins English Dictionary
verb (intr, adverb)1. to become established ⇒ the winter has set in2. (of wind) to blow or (of current) to move towards shoreadjective set-in3. (of a part) made separately and then added to a larger whole ⇒ a set-in sleeve
