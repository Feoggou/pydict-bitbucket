reference number

(ˈrɛfərəns ˈnʌmbə)
Definitions
nouna unique number on an order, application, etc, used to designate and quickly locate that order, application, etc ⇒ Make a note of the reference number shown on the form.
Collins English Dictionary. Copyright © Harper Collins Publishers
