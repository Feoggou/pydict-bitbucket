folder

(ˈfəʊldə )
Definitions
noun1. a binder or file for holding loose papers, etc2. a folded circular3. a machine for folding printed sheets4. a person or thing that folds5. computing another name for directory (sense 5)
Collins English Dictionary. Copyright © Harper Collins Publishers
