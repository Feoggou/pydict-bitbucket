oxalic acid

(ɒkˈsælɪk )
Definitions
nouna colourless poisonous crystalline dicarboxylic acid found in many plants: used as a bleach and a cleansing agent for metals. Formula: (COOH)2. Systematic name: ethanedioic acid
Collins English Dictionary. Copyright © Harper Collins Publishers

Word Origin

C18: from French oxalique, from Latin oxalis garden sorrel; see oxalis
