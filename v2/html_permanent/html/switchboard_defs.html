switchboard

(ˈswɪtʃˌbɔːd )

Definitions

Collins English Dictionary
noun1. an installation in a telephone exchange, office, hotel, etc, at which the interconnection of telephone lines is manually controlled2. an assembly of switchgear for the control of power supplies in an installation or building
