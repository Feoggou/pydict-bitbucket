unregenerate

(ˌʌnrɪˈdʒɛnərɪt )
Definitions
adjective also: unregenerated1. unrepentant; unreformed2. obstinately adhering to one's own viewsnoun3. an unregenerate person
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

unregeneracy (ˌunreˈgeneracy) noununregenerately (ˌunreˈgenerately) adverb
