recurrent

(rɪˈkʌrənt )
Definitions
adjective1. happening or tending to happen again or repeatedly2. anatomy (of certain nerves, branches of vessels, etc) turning back, so as to run in the opposite direction
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms
