coded

(ˈkəʊdɪd)
Definitions
Collins English Dictionary
adjective1. written in code ⇒ In a coded telephone warning, Scotland Yard were told four bombs had been planted in the area., ⇒ a coded message2. not explicit; veiled ⇒ They have sent coded messages to the Education Secretary endorsing this criticism., ⇒ It's widely assumed that his lyrics were coded references to homosexuality.3. electronics using a binary system of digits ⇒ The coded signal is received by satellite dish aerials.

Definitions
COBUILD Advanced British English Dictionary
1. adjective [usu ADJ n]Coded messages have words or symbols which represent other words, so that the message is secret unless you know the system behind the code. ⇒ In a coded telephone warning, Scotland Yard were told four bombs had been planted in the area.2. adjective [usu ADJ n]If someone is using coded language, they are expressing their opinion in an indirect way, usually because that opinion is likely to offend people. ⇒ They have sent barely coded messages to the Education Secretary endorsing this criticism. ⇒ It's widely assumed that his lyrics were coded references to homosexuality.3. adjective [ADJ n]Coded electronic signals use a binary system of digits which can be decoded by an appropriate machine. [technical] ⇒ The coded signal is received by satellite dish aerials.
