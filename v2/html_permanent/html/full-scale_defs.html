full-scale

Definitions
noun (modifier)1. (of a plan, etc) of actual size; having the same dimensions as the original2. done with thoroughness or urgency; using all resources; all-out

Definitions
1. adjective [ADJ n]Full-scale means as complete, intense, or great in extent as possible. ⇒ ...the possibility of a full-scale nuclear war.2. adjective [ADJ n]A full-scale drawing or model is the same size as the thing that it represents. ⇒ ...working, full-scale prototypes.
