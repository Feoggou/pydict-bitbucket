longevity

(lɒnˈdʒɛvɪtɪ )
Definitions
noun1. long life2. relatively long duration of employment, service, etc
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

longevous (lɒnˈdʒiːvəs ) adjective

Word Origin

C17: from Late Latin longaevitās, from Latin longaevus long-lived, from longus long1 + aevum age
