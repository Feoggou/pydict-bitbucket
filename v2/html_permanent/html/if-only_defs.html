Definitions
subordinating conjunction phrase & phraseYou use if only with past tenses to introduce what you think is a fairly good reason for doing something, although you realize it may not be a very good one. ⇒ She writes me often, if only to scold me because I haven't written to her. ⇒ A one-to-one meeting with the U.S. President was necessary, if only for a deeper exchange of views. →  if
