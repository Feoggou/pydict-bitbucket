jet set

or jet-set
Definitions
nouna. a rich and fashionable social set the members of which travel widely for pleasureb. (as modifier) ⇒ jet-set travellers
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

jet-set

(ˈdʒɛtsɛt)
Definitions
modifierof or relating to a rich and fashionable social set the members of which travel widely for pleasure ⇒ the jet-set crowd, ⇒ his hectic jet-set lifestyle
