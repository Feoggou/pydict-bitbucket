chattel

(ˈtʃætəl )
Definitions
Collins English Dictionary
noun1. (often plural) property lawa. See chattel personalb. See chattel real2. See goods and chattels
Word origin
C13: from Old French chatel personal property, from Medieval Latin capitāle wealth; see capital1
