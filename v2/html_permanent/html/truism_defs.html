Definitions
countable nounA truism is a statement that is generally accepted as obviously true and is repeated so often that it has become boring. ⇒ Orpington seems an example of the truism that nothing succeeds like success. ⇒ Whilst this might sound like a truism, it is nevertheless a crucial problem to address.
