successor

(səkˈsɛsə )
Definitions
Collins English Dictionary
noun1. a person or thing that follows, esp a person who succeeds another in an office2. logic the element related to a given element by a serial ordering, esp the natural number next larger to a given one. The successor of n is n + 1, usually written Sn or n′

Derived Forms

successoral (sucˈcessoral) adjective
