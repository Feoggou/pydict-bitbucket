ostensible

(ɒˈstɛnsɪbəl )
Definitions
adjective1. apparent; seeming2. pretended
Collins English Dictionary. Copyright © Harper Collins Publishers

Derived Forms

ostensibility (osˌtensiˈbility) noun

Word Origin

C18: via French from Medieval Latin ostensibilis, from Latin ostendere to show, from ob- before + tendere to extend

ostensible for learners of English

Powered by COBUILD
(ɒstensɪbəl )
Definitions
adjective [ADJ n]Ostensible is used to describe something that seems to be true or is officially stated to be true, but about which you or other people have doubts. [formal] ⇒ The ostensible purpose of these meetings was to gather information on financial strategies.ostensibly (ɒstensɪbli ) adverb [usu ADV with cl/group] ⇒ ...ostensibly independent organisations. ⇒ A bachelor farmer began to call, ostensibly to talk to her father, but really to see her.
