puss1

(pʊs )
Definitions
Collins English Dictionary
noun1. an informal name for a cat1 (sense 1). See also pussy1 (sense 1)2. slang a girl or woman3. an informal name for a hare

Word Origin

C16: related to Middle Low German pūs, Dutch poes, Lithuanian puz
