pitch

verb

  1. =  throw, cast, chuck (informal), fling, heave, hurl, lob (informal), sling, toss
  2. =  set up, erect, put up, raise, settle
  3. =  fall, dive, drop, topple, tumble
  4. =  toss, lurch, plunge, roll

noun

  1. = sports field, field of play, ground, park
  2. =  level, degree, height, highest point, point, summit
  3. =  slope, angle, dip, gradient, incline, tilt
  4. =  tone, modulation, sound, timbre
  5. = sales talk, patter, spiel (informal)
