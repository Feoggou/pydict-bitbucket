affluence (ˈæfluəns ;  (occas.,) æˈfluəns ;  əˈfluəns )

  (ˈæfluənsi ) 

Definitions

noun

  1. a flowing toward; influx
  2. great plenty; abundance
  3. an abundance of riches; wealth; opulence
(Also ) ˈaffluency   (ˈæfluənsi ) 

Word Origin

< L affluentia < affluere, to flow to < ad-, to + fluere, to flow

Synonyms

View thesaurus entry
= wealth, abundance, fortune, opulence, plenty, prosperity, riches
