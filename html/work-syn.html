work

noun

  1. =  effort, drudgery, elbow grease (facetious), exertion, industry, labor, sweat, toil
  2. =  employment, business, duty, job, livelihood, occupation, profession, trade
  3. =  task, assignment, chore, commission, duty, job, stint, undertaking
  4. =  creation, achievement, composition, handiwork, opus, piece, production

verb

  1. =  labor, drudge, exert oneself, peg away, slave, slog, slog away, sweat, toil
  2. = be employed, be in work
  3. =  operate, control, drive, handle, manage, manipulate, move, use
  4. =  function, go, operate, run
  5. =  cultivate, dig, farm, till
  6. =  manipulate, fashion, form, knead, mold, shape
